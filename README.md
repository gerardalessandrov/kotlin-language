Hello World - Kotlin Exploration
Overview
This project is a professional introduction to the Kotlin programming language, developed as part of my Software Development coursework at Brigham Young University-Idaho. It serves as a foundational step to demonstrate environment configuration, basic syntax, and the use of collection types in Kotlin.

Execution Instructions
To run this software locally:

Clone the repository to your local machine.

Open the project folder in IntelliJ IDEA.

Locate the src/HelloWorld.kt file.

Click the Run button (green arrow) or use the shortcut Shift + F10.
